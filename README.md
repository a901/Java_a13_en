# 原始人繁衍模擬器(Java視窗遊戲)

這是我大學時期，出於興趣獨立開發的 Java 視窗遊戲(非學校作業)。專案採用物件導向設計並導入多執行緒技術，遊戲的核心為模擬幾十到幾千個原始人在地圖上覓食與繁衍。

## 核心功能:
隨機生成地圖： 生成地形高低不平，存在山丘與湖泊的地圖，原始人無法攀爬陡坡，極少主動游泳，如此會提高他們覓食的難度。

互動環境： 玩家可手動改變地形，主動增添食物，觀察個體對環境變遷的適應，例如製造迷宮，讓原始人用bfs算法在迷宮中尋找食物。

社會結構模擬： 原始人人口提高後，會進化出部落型態(原始人會採集食物到部落據點，兩個部落間可能戰爭)，並且出現犯罪者(謀殺其他原始人，減低自己的飢餓值)

血緣家庭紀錄: 可以查詢每個原始人的親屬關係，包含父母子女等近親，歷史情人名單，延伸查詢祖輩與後裔

精細行為控制： 具體實作每個原始人，在每個時間單位下的精細行為（移動、吃東西、採集搬運食物、繁衍、攻擊他人）。

## 專案回顧

以我現在（碩士階段）的眼光回視，本專案仍有以下改進空間

記憶體配置過於浪費: 我幾年前在寫專案時，用了大量靜態記憶體配置，應當改用動態配置優化記憶體。

效能層面：專案中的bfs等算法可以再進一步優化。

變數與函式命名: 應該改用更具辨識度的名稱，並增加註解。

## 操作說明

遊戲中除了開始遊戲畫面，之後都是全鍵盤操作(區分大小寫)

方向鍵: 移動游標/所在地圖位置 (按Q改變方向鍵功能)
r:放置岩塊(改變地形，原始人無法一次走高於兩格的山坡)
Y:放置小麥(食物)
H:治癒游標鎖定的原始人(如果他們生病)
t/T:雷擊(消除岩塊，改變地形)
w/W:放置湖泊(原始人通常不會冒險游泳，存在溺死概率)

I: 標記特定原始人(標記者頭上會出現光環，之後可以直接查閱他們的資料，移動到他們所在的位置)
S: 追蹤標記者(右邊的文字視窗，會出現標記者清單，可以用上下方向鍵移動，按X移動到標記者位置)
F: 查詢游標鎖定的原始人的資料(家庭成員等等)

G: 切換大地圖 (改變地圖解析度(只有解析度最高的地圖可以查訊/影響原始人與地形))
g: 切換右上角輔助地圖
V: 查詢大事記/所有人物清單/人口統計
K: 查詢部落清單
P: 暫停/解除暫停
a: 存檔
